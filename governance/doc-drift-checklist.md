### Doc Drift Checklist
- env vars, config flags
- API request/response, errors
- component states/props (UI)
- migration steps
- performance budgets (if UI)
- security changes (authN/Z, secrets)

### README Coverage Index (0–100)

Threshold default: ≥80 before finalization.
